## Première connexion

### Connexion
1. Accédez à l'URL : `https://inventory.iuc.cm`
2. Entrez votre email et mot de passe
3. Cliquez sur "Connexion"

### Récupération mot de passe
1. Cliquez sur "Mot de passe oublié"
2. Entrez votre email
3. Suivez les instructions reçues par email



## Tableau de bord

Le tableau de bord affiche :
- **KPIs** : Produits en stock, valeur totale, commandes en cours, alertes
- **Graphiques** : Consommation mensuelle, répartition par catégorie
- **Activités récentes** : Derniers mouvements de stock

### Navigation
- Utilisez la barre latérale gauche pour naviguer
- La barre de recherche en haut permet de trouver rapidement
- Les notifications apparaissent en haut à droite



## Gestion des produits

### Créer un produit
1. Allez dans **Produits** > **Nouveau produit**
2. Remplissez les champs obligatoires :
   - Nom
   - Référence (unique)
   - SKU
   - Catégorie
   - Prix unitaire
3. Cliquez sur **Créer**

### Modifier un produit
1. Cliquez sur le produit dans la liste
2. Modifiez les champs souhaités
3. Cliquez sur **Enregistrer**

### Rechercher un produit
- Utilisez la barre de recherche
- Filtrez par catégorie, fournisseur, statut de stock



## Gestion des stocks

### Vue d'ensemble
La section **Stocks** affiche :
- Stock actuel par entrepôt
- Quantités disponibles et réservées
- Emplacements
- Valeur totale

### Effectuer une entrée de stock
1. Allez dans **Stocks** > **Entrées**
2. Cliquez sur **Nouvelle entrée**
3. Sélectionnez l'entrepôt et le fournisseur
4. Ajoutez les produits et quantités
5. Validez l'entrée

### Effectuer une sortie de stock
1. Allez dans **Stocks** > **Sorties**
2. Cliquez sur **Nouvelle sortie**
3. Sélectionnez le motif (usage interne, transfert...)
4. Ajoutez les produits et quantités
5. Validez la sortie

### Transfert entre entrepôts
1. Allez dans **Stocks** > **Transferts**
2. Sélectionnez l'entrepôt source et destination
3. Ajoutez les produits à transférer
4. Validez le transfert



## Commandes fournisseurs

### Créer une commande
1. Allez dans **Commandes** > **Nouvelle commande**
2. Sélectionnez le fournisseur
3. Ajoutez les produits et quantités
4. Soumettez pour approbation

### Suivre une commande
- **Brouillon** : En cours d'édition
- **En attente** : Soumise, en attente d'approbation
- **Approuvée** : Validée, en attente de livraison
- **Reçue** : Livrée, entrée en stock créée



## Inventaires

### Créer un inventaire
1. Allez dans **Inventaires** > **Nouvel inventaire**
2. Sélectionnez l'entrepôt
3. Les lignes sont générées automatiquement depuis le stock actuel

### Effectuer le comptage
1. Ouvrez l'inventaire créé
2. Pour chaque ligne, entrez la quantité comptée
3. Les écarts sont calculés automatiquement

### Valider l'inventaire
1. Vérifiez les écarts
2. Cliquez sur **Valider l'inventaire**
3. Les stocks sont mis à jour automatiquement



## Rapports

### Générer un rapport
1. Allez dans **Rapports**
2. Sélectionnez le type de rapport :
   - Niveau de stock
   - Mouvements
   - Consommation par département
   - Inventaire
3. Choisissez le format (PDF, Excel, CSV)
4. Définissez les filtres (dates, entrepôt...)
5. Cliquez sur **Générer**

### Téléchargement
Le rapport est téléchargé automatiquement dans le format choisi.



## QR Codes

### Générer un QR code
1. Allez dans un produit
2. Cliquez sur l'icône QR Code
3. Le QR code est généré automatiquement

### Scanner un QR code
1. Allez dans **QR Codes**
2. Entrez le contenu du QR code
3. Les informations du produit s'affichent

### Impression en masse
1. Sélectionnez plusieurs produits
2. Cliquez sur **Imprimer QR Codes**
3. Une planche A4 est générée



## Assistant IA

### Poser une question
1. Cliquez sur l'icône **Assistant IA** (bulle en bas à droite)
2. Tapez votre question
3. L'assistant répond avec des suggestions

### Exemples de questions
- "Combien reste-t-il d'ordinateurs Dell ?"
- "Quels produits sont en rupture ?"
- "Quel département consomme le plus ?"
- "Comment créer une entrée de stock ?"
